React Samples for MindFusion.Scheduling for JavaScript, version 1.1


1. Create React App

	npm install -g create-react-app

	create-react-app jsplanner-samples

	cd jsplanner-samples


2. Install MindFusion.Scheduling for JavaScript

	npm install scheduler-library


3. Copy the contents of Samples\React folder into jsplanner-samples folder


4. Run the sample application

	npm start

About this sample

This sample demonstrates the grouping and filtering capabilities of JsPlanner.
Filtering is used to display calendar items, associated with a selected resource and can be enabled by setting the GroupType property to one of the FilterBy* values. Select a contact or location from one of the drop-down lists below in order to display the matching appointments.
Grouping the calendar creates separate rows or columns for each resource and can be enabled by setting the GroupType property to one of the GroupBy* values. Note that the Resource view is designed to only display content when grouping is enabled.

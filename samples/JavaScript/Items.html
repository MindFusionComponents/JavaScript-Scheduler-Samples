About this sample

This sample shows how to create calendar items.
Calendar items represent events scheduled to occur at an appointed time and are defined by instances of the Item class. Items can have associated resources via the resources, contacts, location and task properties. Items can recur in time, following a specific recurrence pattern. The appearance of calendar items can be customized by using the calendar.itemSettings property or by specifying a css class to individual items.
JSplanner offers out-of-the-box interactive creating, modifying and deleting of calendar items by using the built-in popup forms.
